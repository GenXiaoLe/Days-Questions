## 前端设计模式学习总结

### 单例模式
> 单例模式顾名思义只有一个实例可以在全局使用，实现方式有多种

1. 挂在window上(不推荐)
单例可以把实例挂在window上，这样在全局都可以实现调用，但是由于会有数据污染的风险 所以不是很推荐

```
class A {
  ...
}

window.__VM__._A = new A()
```

2. 使用闭包封装全局实例
使用闭包可以实现实例的缓存，这样只需要在需要调用的地方export即可

```
class A {
  ...
}

const _ProxyCreate = (function () {
  let instance

  return function () {
    // 如果实例存在，直接导出
    if (instance) {
      return instance
    }

    // 否则创建实例导出
    return instance = new A()
  }
})()


export {
  _A: new _ProxyCreate()
}
```
