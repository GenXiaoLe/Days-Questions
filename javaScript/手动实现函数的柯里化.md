## 实现函数柯里化

### 基础用法
```
const add = (a, b, c, d) => a + b + c + d

const fn = curry(add)

fn(1)(2)(3)(4) // 10
```


### 柯里化实现
```
function curry (fn) {
  // 当fn的数量只有一个之后 直接返回
  if (fn.length <= 1) return fn

  const generator = (...args) => {
    // 如果fn的长度等于入参长度 说明是最后一次 执行函数
    if (fn.length === args.length) {
      return fn(...args)
    }
    
    // 否则把入参合并 继续递归走下去
    return (...args2) => {
      return generator(...args, ...args2)
    }
  }

  return generator
}
```
